### Install
* make sure you have [ruby, jekyll and bundler installed](https://help.github.com/articles/using-jekyll-with-pages/)
* `npm install`
* `bower install`
* `grunt`
* to run locally `bundle exec jekyll serve`
* to watch and compile /app/ files `grunt watch`
